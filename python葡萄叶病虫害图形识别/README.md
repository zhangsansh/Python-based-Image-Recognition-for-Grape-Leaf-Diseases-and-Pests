# 基于深度学习的葡萄叶病虫害图形识别系统

## 项目描述

本项目致力于构建一个基于Web的葡萄叶病虫害智能识别系统。系统利用Python Flask框架开发后端服务，提供Web前端界面供用户上传葡萄叶图像。核心功能是采用先进的深度学习模型（EfficientNet, MobileNetV3）对上传的叶片图像进行自动识别，判断其健康状况或所感染的病害类型（如霜霉病、白粉病、黑腐病等）。

## 应用场景

1. **智慧农业**：为葡萄种植户、农技人员提供快速、便捷的田间病虫害诊断工具，辅助早期发现和精准防治，减少农药滥用，提高葡萄品质与产量。
2. **农业科研与教育**：辅助植物病理学研究，作为农业院校教学实践的案例工具。
3. **农业咨询服务**：集成到农业APP或服务平台中，提供在线的植物医生功能。

## 技术栈

- **后端框架**：Flask
- **深度学习模型**：EfficientNet, MobileNetV3
- **图像处理**：OpenCV, PIL
- **前端技术**：HTML, CSS, JavaScript, Bootstrap

## 安装指南

### 环境准备

1. 确保已安装Python 3.8或更高版本
2. 克隆本项目代码

### 安装依赖

```bash
pip install -r requirements.txt
```

## 使用方法

### 启动服务

```bash
python app.py
```

### 访问系统

打开浏览器，访问 http://localhost:5000

## 项目结构

```
├── app.py              # 主应用入口
├── requirements.txt    # 依赖包列表
├── models/             # 存储训练好的模型
├── static/             # 静态资源文件
│   ├── css/            # 样式文件
│   ├── js/             # JavaScript文件
│   └── images/         # 图片资源
├── templates/          # HTML模板
├── utils/              # 工具函数
└── data/               # 数据集和预处理脚本
```

## 功能说明

1. **图像上传**：用户可以上传葡萄叶图像文件
2. **图像预处理**：系统自动对上传图像进行预处理（缩放、归一化等）
3. **病害识别**：使用预训练的深度学习模型进行分类识别
4. **结果展示**：显示识别结果、置信度及防治建议

## 数据来源

- AI Challenger 2018（葡萄叶）
